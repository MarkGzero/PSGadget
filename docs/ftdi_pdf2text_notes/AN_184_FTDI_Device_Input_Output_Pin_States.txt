1 Introduction

This application note explains the various states of input and output pins of the following FTDI devices:

FT232R, FT245R, FT232RN, FT245RN, FT232H, FT2232H, FT2232H-56Q, FT4232H, FT4232H-56Q,
FT232HP, FT233HP, FT2232HP, FT2233HP, FT4232HP, FT4233HP,  FT2232D, FT200XD,  FT201X,
FT220X, FT221X, FT230X, FT231X, FT234XD, FT240X, FT120, FT121, FT122, FT313H, FT4222H and
FT260.

Note: The convention used throughout this document for active low signals is the signal name followed
by a #.
